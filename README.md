# E-Commerce Customer Behavior & Revenue Analytics Using SQL

## Project Overview

This project analyzes 25,000 e-commerce customer sessions using MySQL. The goal is to understand customer behavior, conversion performance, revenue generation, cart abandonment trends, and marketing effectiveness.

## Dataset Information

- Total Records: 25,000
- Unique Customers: 8,442
- Dataset Type: E-commerce Customer Behavior
- Tools Used: MySQL Workbench, SQL, Excel

## Business Questions Solved

1. What is the total revenue generated?
2. How many unique customers visited the platform?
3. What is the conversion rate?
4. What is the cart abandonment rate?
5. Which product categories generate the highest revenue?
6. Which device types generate the most revenue?
7. Which marketing channels perform best?
8. Who are the top revenue-generating customers?
9. How can customers be segmented based on revenue?

## Key Insights

- Generated over ₹10.1 million in revenue.
- Analyzed behavior of 8,442 unique customers.
- Conversion rate was 22.46%.
- Cart abandonment rate was 42%, indicating significant drop-off before purchase.
- Device Type 1 generated the highest revenue contribution.
- Marketing Channel 5 generated the highest revenue among all channels.
